# Управление магазином (для владельца)

Весь каталог магазина берётся из **одной Google-таблицы**. Вы добавляете/меняете
строки в таблице — и сайт обновляется. Не нужно трогать код.

> Технически: `npm run catalog` читает таблицу → создаёт `src/catalog.json` и
> `src/categories.json` → `npm run build` собирает сайт. Если настроено
> авто-обновление (GitHub Actions, см. ниже), всё происходит само.

---

## Колонки таблицы

| Колонка | Обязательно | Что это |
|---|---|---|
| **Brand** | желательно | Бренд (DNK'a, DANNY, Staleks Pro…). Пусто → товар попадёт в группу «Fără brand». |
| **SKU** | желательно | Артикул, напр. `T1338`. **Должен быть уникальным** — по нему строится ссылка на товар и имя файла фото. |
| **Category** | да | Раздел (Culori, Baze, Topuri…). Новый текст = новый раздел появится сам. |
| **Title** | да | Название товара (на румынском). RU-название генерируется автоматически. |
| **Text** | — | Описание. |
| **Price** | да | Цена (число, лей). Без цены строка игнорируется. |
| **Price Old** | — | Старая цена. Если больше Price — покажется скидка и значок «−%». |
| **Photo** | — | Прямая ссылка на фото (см. ниже). |
| **Characteristics:Cantitate** | — | Объём (15ml, 30ml…). |
| **Characteristics:Culoare** | — | Цвет (Roz, Negru…). |
| **Characteristics:Putere** | — | Мощность (65 W…). |
| **Characteristics:Viteză maximă** | — | Скорость (35.000 rpm…). |
| любая **Characteristics:…** | — | Любая своя колонка `Characteristics:НазваниеХарактеристики` появится в карточке товара. |

## Как…

**Добавить товар** — новая строка: заполните Category, Title, Price (минимум).
Дайте уникальный SKU.

**Добавить фото** — загрузите картинку на [Cloudinary](https://cloudinary.com)
(бесплатно) или любой хостинг, скопируйте **прямую ссылку на файл** (заканчивается
на `.jpg`/`.png`/`.webp`) и вставьте в колонку **Photo**. Без watermark — это ваши
файлы. Можно и без Photo: тогда покажется аккуратная цветная заглушка.
Альтернатива: положить файл `<SKU>.jpg` в `public/images/products/`.

**Добавить бренд** — просто впишите его в колонку Brand. Он сам появится в фильтрах
и в блоке брендов.

**Добавить раздел** — впишите новый текст в Category. Раздел появится автоматически
со стандартным оформлением. (Чтобы задать красивое название RU/RO, цвет и иконку —
добавьте строку в `CURATED_CATS` в `src/data.js`.)

**Сделать скидку** — впишите старую цену в Price Old (больше текущей Price).

## Чтобы изменения попали на сайт

После правок в таблице нажмите **меню «🚀 Nail Mania» → «Опубликовать на сайте»** —
сайт пересоберётся и обновится за ~1–2 минуты. (Сайт статический, поэтому правки
видны не сразу, а после публикации.) Настройка кнопки — в [PUBLISH.md](PUBLISH.md).

## Разовая настройка (для разработчика)

1. Импортировать `nailmania-sheet.csv` в Google Sheets (File → Import → Replace spreadsheet).
2. File → Share → **Publish to web** → выбрать нужный лист → формат **CSV** → копировать ссылку.
3. Положить ссылку в переменную окружения `CATALOG_SHEET_URL`
   (локально или в Secrets репозитория для GitHub Actions).
4. Проверить: `CATALOG_SHEET_URL="<ссылка>" npm run catalog` — должно вывести
   «source: Google Sheet».
