# Приём заказов в Telegram

Когда покупатель оформляет заказ, сайт отправляет его в Telegram владельцу.
Токен бота хранится **только на сервере** (в секрете), а не в коде сайта — в
исходниках страницы его не видно.

Как это устроено: фронтенд шлёт заказ на `/api/order`, а дальше —
- **Cloudflare Pages** → функция [`functions/api/order.js`](../functions/api/order.js)
- **iphost / cPanel** → [`order.php`](../public/order.php) (на него `.htaccess` перенаправляет `/api/order`)

---

## 0. Безопасность — сначала
Токен, который был отправлен в переписке, считается скомпрометированным.
Откройте **@BotFather → /revoke**, получите **новый** токен и используйте только его.
Нигде не вписывайте токен в код, который попадает в git.

## 1. Узнать chat_id (куда слать заказы)
1. Напишите своему боту любое сообщение в Telegram (нажмите Start).
2. Откройте в браузере: `https://api.telegram.org/bot<ТОКЕН>/getUpdates`
3. Найдите `"chat":{"id":...}` — это и есть **chat_id** (для личных сообщений —
   ваш id; для группы — отрицательное число). Можно слать и в группу: добавьте
   туда бота и возьмите id группы.

## 2. Подключение

### Вариант A — Cloudflare Pages (рекомендуется)
В панели проекта Pages → **Settings → Environment variables** добавьте:
- `TELEGRAM_BOT_TOKEN` = новый токен
- `TELEGRAM_CHAT_ID` = ваш chat_id

Передеплойте. Готово — функция `/api/order` заработает сама.

### Вариант B — iphost / cPanel (PHP)
1. На сервере в `public_html` создайте файл `telegram.config.php`:
   ```php
   <?php
   return [
     'token'   => 'НОВЫЙ-ТОКЕН',
     'chat_id' => 'ВАШ-CHAT-ID',
   ];
   ```
   (Шаблон — `telegram.config.example.php`. Реальный файл в git не коммитится.)
2. Убедитесь, что на хостинге включён PHP (на cPanel — да).
   `.htaccess` уже перенаправляет `/api/order` → `order.php`.

## 3. Проверка
Оформите тестовый заказ на сайте — сообщение должно прийти в Telegram.
Если не пришло: проверьте chat_id, что боту написали Start, и значения секретов.

> Заказ всегда дублируется в браузере покупателя (история), а оформление
> срабатывает, даже если Telegram временно недоступен — продажа не теряется.
